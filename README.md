# Metatranscriptomic Analysis of *Bacteroides xylanisolvens*

## Project Overview
This repository contains a specialized bioinformatic pipeline for the **metatranscriptomic** analysis of *Bacteroides xylanisolvens* isolated from bovine diarrhea samples in Iraq (Wasit Province).

## Author
**Ibrahim H. Madhloom** *PhD in Microbiology (Viruses)* University of Basra, College of Veterinary Medicine

## Pipeline Stages
The included script (`B_xylanisolvens_Metatranscriptome_Pipeline.py`) automates the following:
1. **Environment Setup:** Installing Conda, Prokka, and BLAST dependencies.
2. **Data Recovery:** Processing genomic archives from Google Drive.
3. **Strain Extraction:** Bulk processing of 75 strains and 6.4kb genomic fragments.
4. **Functional Annotation:** Gene prediction and characterization.

## Results Summary
The analysis identifies active microbial pathways and provides functional insights into the bovine gut microbiome during infection.
